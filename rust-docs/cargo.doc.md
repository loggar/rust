# cargo doc

```
cargo doc --open
```

This will build the HTML for your current crate’s documentation (as well as the documentation for all of your crate’s dependencies).

## Document styles

````rust
/// Adds one to the number given.
///
/// # Examples
///
/// ```
/// let arg = 5;
/// let answer = my_crate::add_one(arg);
///
/// assert_eq!(6, answer);
/// ```
pub fn add_one(x: i32) -> i32 {
    x + 1
}
````
